# 🚀 SkillSwap - Skills Exchange Mobile App

**SkillSwap** is a professional Android application designed for users to exchange skills with each other. This project is developed as part of a Mobile Application Development (MAD) course assignment.

---

## 🎯 Project Objective
The goal of this application is to create a community where people can teach what they know and learn what they need through a "Skill Swap" request system.

---

## ✨ Features (Phase 1 to 4 Complete)
Current implementation includes the full user-side frontend flow:

* **Splash Screen:** Professional app entry with logo.
* **Authentication:** * Clean Login UI with validation.
    * Signup screen for new user registration.
* **Skill Listing (Home):** * Used **RecyclerView** and **Custom Adapter** to display available skills.
    * Dynamic card layout using **CardView**.
* **Detailed View:** Detailed information about each skill and the teacher.
* **Request System:** * Interactive **Request Form** for skill swapping.
    * Input validation and success feedback via **Toasts**.
* **My Requests:** A list of sent requests using **ListView**.

---

## 🛠️ Technical Stack (Frontend Only)
* **Language:** Java
* **UI Design:** XML (Layouts, Drawables, Styles)
* **Navigation:** Intents & Activity Transition
* **Key Widgets Used:** * RecyclerView, CardView, ListView, ScrollView, LinearLayout.
    * Material Buttons, EditText (Forms), Toasts.

---

## 📁 Project Structure
- `Java/com.example.skillswap`: Contains all logic (Activities, Models, Adapters).
- `res/layout`: Contains all XML design files.
- `manifests`: App configuration and screen registration.
